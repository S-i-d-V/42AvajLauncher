# UML Diagram:

## UML Symbols :
### Class symbols meanings :
<ul>
    <li>+ : Public</li>
    <li>- : Private</li>
    <li># : Protected</li>
    <li>~ : Package</li>
</ul>

### Relations arrows meanings :
<ul>
    <li>→ : Inheritance</li>
    <li>⇢ : Dependancies</li>
    <li>◇ : Aggregation</li>
    <li>◆ : Composition</li>
</ul>
